# Frontend - React

## Установка

1. Установи зависимости:
   ```bash
   npm install
   ```

2. Запусти разработческий сервер:
   ```bash
   npm start
   ```

Приложение откроется на `http://localhost:3000`

## Структура проекта

```
src/
├── App.js              # Главный компонент
├── App.css             # Стили главного компонента
├── index.js            # Точка входа
├── index.css           # Глобальные стили
└── components/
    ├── UserForm.js     # Форма добавления пользователя
    ├── UserForm.css    # Стили формы
    ├── UserList.js     # Список пользователей
    └── UserList.css    # Стили списка
```

## Функциональность

- Просмотр всех пользователей
- Добавление новых пользователей
- Редактирование пользователей
- Удаление пользователей
- Красивый и отзывчивый интерфейс
